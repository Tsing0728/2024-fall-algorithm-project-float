运行环境
1、使用Visual Studio 
2、保证两台计算机在同一局域网下，且IP地址不可为100.xx.xx.xx或10.xx.xx.xx。

程序运行说明：
1、共有单机版程序、单机加速版程序与双机加速版程序共四个文件，需要分别运行。
2、单机版程序、单机加速版程序可直接运行，无需任何修改。
3、双机加速版程序在运行前需要做修改。在sender的60行设置服务器的地址信息（现在写的是127.0.0.1）（先开receiver，别着急，等两秒再开sender，不然可能显示收发字节数不正确，但对结果没有影响）

注：1.排序检查函数checkSort的返回值为1代表排序正确，为0代表排序错误。
